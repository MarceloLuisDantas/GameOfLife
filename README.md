# GameOfLife
Interative Conway's Game of Life writing in C

![Alt Text](demo.gif)

## Commands
### On Pause Menu
| Key    | Action   |
| ------ | -------- |
| Arrows | Move the Cursor |
| x      | Switch the state of a cell |
| n      | Next Generation |
| ENTER  | Resume the Simulation |

### On Simulation
| Key    | Action  |
| ------ | --------|
| -      | Slows the simulation |
| +/=    | Speeds up the simulation |
| ENTER  | Pause the the simulation |
